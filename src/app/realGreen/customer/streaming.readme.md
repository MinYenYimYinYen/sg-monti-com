| Concept / Entity                         | What It Is                                                                              | Why It's Here (The Mechanism)                                                                                                                                                                                                               |
|:-----------------------------------------|:----------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **`ReadableStream`**                     | A standard Web API object representing a source of data that can be read incrementally. | **The Core Vehicle.** Instead of waiting for *all* data to be ready (like a standard JSON response), this object allows the server to start sending data immediately and keep the connection open to send more later.                       |
| **`start(controller)`**                  | The initialization method of the `ReadableStream`.                                      | **The Engine Room.** This is where your business logic lives. It runs asynchronously. The `controller` argument is your tool to "push" data into the stream.                                                                                |
| **`controller`**                         | Specifically `ReadableStreamDefaultController`.                                         | **The Interface.** It provides the methods to interact with the stream: `enqueue()` to send data, `close()` to finish, and `error()` to kill the stream.                                                                                    |
| **`TextEncoder`**                        | A utility that converts JavaScript strings into `Uint8Array` (binary data).             | **The Translator.** Streams transport *bytes*, not strings or objects. You cannot `enqueue` a string directly; you must encode it into bytes first.                                                                                         |
| **`controller.enqueue(...)`**            | A method on the controller.                                                             | **The "Send" Button.** This pushes a chunk of data down the wire to the client. In your case, you are pushing encoded JSON strings.                                                                                                         |
| **NDJSON (`\n`)**                        | Newline Delimited JSON. A format where each line is a valid JSON object.                | **The Protocol.** Since we are sending multiple JSON objects over one connection, we need a way to tell where one ends and the next begins. The newline character `\n` acts as this separator.                                              |
| **`Content-Type: application/x-ndjson`** | An HTTP Header.                                                                         | **The Handshake.** It tells the browser/client, "I am not sending you one big JSON file. I am sending you a stream of newline-separated JSON objects."                                                                                      |
| **`Transfer-Encoding: chunked`**         | An HTTP Header.                                                                         | **The Transport Mode.** It tells the network layer, "I don't know the total size of this response yet. I'll send it in chunks as I generate it." (Next.js/Node often handles this automatically for streams, but it's good to be explicit). |
| **`AsyncGenerator`** (Implied)           | A function that yields promises (e.g., `async function*`).                              | **The Data Source.** While not strictly required by the stream itself, this is the standard pattern for *producing* data lazily in your business logic so the stream can consume it one step at a time.                                     |
